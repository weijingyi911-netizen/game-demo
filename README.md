# 可视化剧情编辑器 (Visual Story Editor)

一个基于 Web 的可视化剧情编辑工具，专为独立游戏开发者和剧情策划设计。支持节点式编辑、实时预览、资源管理和 YAML 数据导出。

## 核心功能

- **可视化节点编辑**：
  - 支持 **对话**、**选项**、**条件**、**结局** 四种核心节点。
  - 拖拽式连接，直观管理剧情分支。
  - 右侧属性面板，实时编辑节点详情（说话人、背景、立绘、跳转逻辑等）。
  - **复制节点**：一键复制现有节点，快速复用模板。

- **章节管理**：
  - 多章节支持，每个章节拥有独立的节点画布。
  - 章节增删改查，自动保存。

- **资源管理**：
  - 内置基础素材（背景、角色）。
  - 支持上传本地图片资源（Demo 版支持本地模拟上传）。

- **实时预览与导出**：
  - 一键预览当前章节的运行效果。
  - 导出标准的 YAML 数据格式，可直接用于游戏引擎解析。

## 技术栈

- **前端**：原生 HTML5, CSS3 (Stardew Valley Pixel Theme), JavaScript (ES6+)
- **图形渲染**：SVG (连接线绘制), DOM (节点渲染)
- **数据存储**：LocalStorage (本地自动保存)

## 快速开始

1. **启动项目**：
   使用任意静态文件服务器（如 `Live Server`, `http-server`, `python -m http.server`）运行项目根目录。
   
   ```bash
   # 使用 python 启动
   python3 -m http.server 3001
   ```

2. **访问编辑器**：
   打开浏览器访问 `http://localhost:3001/editor.html` 进入编辑器。

3. **预览游戏**：
   在编辑器中点击“预览”，或直接访问 `http://localhost:3001/game.html`。
