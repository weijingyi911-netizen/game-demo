## 修复计划

### 第一阶段：修复 `net::ERR_ABORTED` 错误
1. 检查并修复 `VisualEditorScene.js` 中可能的语法问题
2. 确保 Vite 开发服务器配置正确
3. 验证所有模块导入路径

### 第二阶段：重构游戏界面为组件化架构
**核心改动：将游戏从 Phaser Canvas 改为 HTML 组件**

1. **创建 `game.html`** - 独立的游戏页面
   - 使用纯 HTML/CSS/JS 实现
   - 不依赖 Phaser canvas

2. **创建游戏组件系统** (`src/game/`)
   - `GameEngine.js` - 游戏状态管理引擎
   - `components/DialogueBox.js` - 对话框组件
   - `components/ChoicePanel.js` - 选项面板组件
   - `components/CharacterDisplay.js` - 角色立绘组件
   - `components/BackgroundDisplay.js` - 背景图片组件
   - `components/AffectionBar.js` - 好感度条组件

3. **数据流设计**
   ```
   编辑器 (editor.html)
       ↓ localStorage
   游戏数据 (JSON)
       ↓ 
   游戏组件 (game.html)
   ```

4. **更新编辑器预览功能**
   - 修改 `previewStory()` 跳转到 `game.html`
   - 确保数据格式兼容

### 第三阶段：整合与测试

2. 测试编辑器 → 游戏预览的数据流
