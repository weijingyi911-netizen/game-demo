# 可视化剧情编辑器 - 版本迭代文档

## 项目概述
- **项目名称**: X-project 可视化剧情编辑器
- **技术栈**: Phaser 3 + Vite + HTML/CSS
- **目标**: 为非程序员提供可视化剧情编辑工具

---

## 版本历史

### v0.1.0 - 初始版本 (2024-02-27)
**功能**:
- 基础三栏布局（左侧章节列表、中间画布、右侧属性面板）
- 四种节点类型：对话、选项、条件、结局
- YAML 数据解析和导出

**问题**:
- 节点无法点击选中
- 右侧面板输入框无法交互

---

### v0.1.1 - 交互修复尝试 (2024-02-27)
**修改内容**:
- 添加 depth 层级排序
- 调整元素间距防止重叠

**问题**:
- 整个页面是单个 canvas，无法选择具体元素
- 右侧输入框仍然无法交互
- 语法错误导致页面无法加载

**根本原因**:
- Phaser 使用 Canvas 渲染，所有元素绘制在同一个 canvas 上
- Canvas 内部元素无法像 HTML DOM 那样直接交互
- 需要使用 Phaser 的 DOM 插件或完全使用 HTML/CSS 重写

---

### v0.2.0 - 架构重构 (2024-02-27)
**重大变更**: 使用 HTML/CSS 重写编辑器，放弃 Phaser Canvas

**修改内容**:
- 创建独立的 `editor.html` 文件
- 使用纯 HTML/CSS/JS 实现编辑器界面
- 使用 DOM 元素实现可拖拽节点
- 输入框使用原生 HTML input/textarea/select

**解决的问题**:
- ✅ P001: 节点可以点击选中
- ✅ P002: 输入框可以正常交互
- ✅ P003: 语法错误已修复
- ✅ P004: 可以选择具体元素

**新增功能**:
- 节点拖拽
- 实时属性编辑
- 数据自动保存到 localStorage
- YAML 导出

### v0.3.0 - 节点连线功能 (2024-02-27)
**重大变更**: 添加节点连线功能，数据格式与游戏完全同步

**新增功能**:
- ✅ 节点之间可视化连线
- ✅ SVG 曲线连接线
- ✅ 对话节点 `goto` 属性
- ✅ 选项节点每个选项可独立跳转
- ✅ 条件节点 `true_goto` / `false_goto`
- ✅ 连接点拖拽交互
- ✅ 下拉选择目标节点
- ✅ 节点 ID 可编辑

**数据格式同步**:
- 对话节点: `background`, `character`, `speaker`, `text`, `goto`
- 选项节点: `type: choice`, `options[].text`, `options[].affection`, `options[].goto`
- 条件节点: `type: condition`, `check`, `true_goto`, `false_goto`
- 结局节点: `type: ending`, `ending`

**待实现功能**:
- [ ] 预览功能与游戏场景对接
- [ ] 多章节支持
- [ ] 撤销/重做功能

## 技术决策记录

### 决策 1: 使用 HTML/CSS 重写编辑器
- **时间**: 2024-02-27
- **背景**: Phaser Canvas 无法实现复杂的表单交互
- **决策**: 编辑器部分使用纯 HTML/CSS/JS，游戏预览使用 Phaser
- **影响**: 需要重构整个编辑器代码

### 决策 2: 保持 YAML 数据格式
- **时间**: 2024-02-27
- **背景**: 需要与游戏数据格式保持一致
- **决策**: 继续使用 YAML 作为数据交换格式
- **影响**: 需要实现 YAML 解析和生成

### v0.5.0 - 组件化游戏架构 (2024-03-04)
**重大变更**: 游戏界面从 Phaser Canvas 改为 HTML 组件架构

**问题背景**:
- `net::ERR_ABORTED` 页面加载错误
- 编辑器的 div 修改无法映射到游戏 canvas
- 游戏界面是整幅 canvas，无法组件化配置

**解决方案**:
- 创建独立的 `game.html` 组件化游戏页面
- 创建 `GameEngine.js` 游戏引擎核心
- 游戏界面由多个 HTML 组件构成（背景、角色、对话框、选项等）
- 编辑器数据直接映射到游戏组件

**新增文件**:
- `game.html` - 组件化游戏主页面
- `src/game/GameEngine.js` - 游戏引擎核心

**修改文件**:
- `index.html` - 更新菜单链接
- `editor.html` - 更新预览功能

**解决的问题**:
- ✅ P008: 页面加载错误
- ✅ P009: 编辑器数据无法映射到游戏
- ✅ P010: 游戏界面组件化
